# 🛒 Shopper Spectrum

Customer Segmentation and Product Recommendation System using RFM Analysis and Machine Learning.

## 📌 Project Overview

Shopper Spectrum is a machine learning-based customer analytics application that helps businesses understand customer behavior through customer segmentation and product recommendation techniques.

The project uses RFM (Recency, Frequency, Monetary) analysis to identify customer groups and provide actionable business insights for targeted marketing and customer retention.

---

## 🎯 Problem Statement

E-commerce businesses generate large volumes of customer transaction data. Analyzing this data manually is difficult and often fails to reveal meaningful customer patterns.

This project aims to:

- Segment customers based on purchasing behavior.
- Identify valuable and at-risk customers.
- Generate product recommendations.
- Support data-driven marketing and business decisions.

---

## 📊 Dataset Used

**Online Retail Dataset**

Features used for RFM Analysis:

- Recency
- Frequency
- Monetary Value

## 🤖 Machine Learning Models

### 1. K-Means Clustering
Used for customer segmentation based on RFM features.

### 2. Hierarchical Clustering
Used to create customer groups using hierarchical relationships between customers.

### 3. Gaussian Mixture Model (GMM)
Used for probabilistic customer segmentation.

---

## 📈 Evaluation Metric

### Silhouette Score

Silhouette Score was used to evaluate clustering performance.

It measures:

- Cluster Cohesion
- Cluster Separation

Higher values indicate better customer segmentation quality.

---

## 🔧 Hyperparameter Tuning

Hyperparameter tuning was performed using Silhouette Score optimization.

Different cluster/component values were tested and the configuration with the highest Silhouette Score was selected.
